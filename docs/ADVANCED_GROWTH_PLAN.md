Planificar mejoras avanzadas de growth, monetización y automatización para escalar la app.

- [ ] Implementar A/B testing framework (Firebase Remote Config)

- [ ] Optimizar paywall/premium (si aplica)

- [ ] Habilitar Renovate para updates automáticos de dependencias
