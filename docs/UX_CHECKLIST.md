# 🎨 UX & Accesibilidad Checklist - FurgoKid

## Objetivo

Mejorar la experiencia de usuario y accesibilidad para todos los perfiles.

---

## 📋 Checklist de acciones (pendiente usuario)

### Onboarding & UX

- [ ] Tutorial interactivo para nuevos usuarios (tooltips, walkthrough)
- [ ] Email verification con UX amigable (retry, feedback)
- [ ] Loading states informativos (no solo spinners)
- [ ] Toast/Snackbar para feedback de acciones
- [ ] Botón "Retry" en errores de red

### Accesibilidad (WCAG 2.1)

- [ ] Contraste de colores > 4.5:1 (validar con https://webaim.org/resources/contrastchecker/)
- [ ] Labels de accesibilidad en botones y campos
- [ ] Font scaling respetado (accesibilidad sistema)
- [ ] Navegación por teclado funcional

### Engagement

- [ ] Push notifications con deep linking
- [ ] In-app messaging para features nuevas
- [ ] Gamification (badges, streaks)
- [ ] NPS survey (Net Promoter Score)
