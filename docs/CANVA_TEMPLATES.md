## 1️⃣ OPEN GRAPH IMAGE (1200x630px)

### Crear en Canva:

1. Ir a: https://canva.com/create/facebook-posts/
2. Seleccionar template en blanco
3. Aplicar estas especificaciones:

### BACKGROUND:

```

Dirección: Diagonal superior izquierda → inferior derecha
Color 1: #4A90E2 (Azul claro)
Color 2: #2E7BC4 (Azul oscuro)
```

### ELEMENTOS:

**Logo (Superior Izquierda):**

```

```

**Screenshot Mapa (Centro):**


## ✅ CHECKLIST DE DISEÑO

Antes de descargar CUALQUIER asset de Canva, verifica:

- [ ] Dimensiones correctas (exactas)
- [ ] Colores de paleta FurgoKid
- [ ] Fuente Montserrat (o alternativa aprobada)
- [ ] Logo visible y legible
- [ ] Contraste suficiente (texto vs fondo)
- [ ] Sin elementos cortados en bordes
- [ ] Calidad de export: Alta
- [ ] Formato: PNG (no JPG para imágenes con texto)
- [ ] Nombre de archivo correcto
