Hola! Necesito que me guíes paso a paso para crear 3 tipos de diseños en Canva Pro para mi app "FurgoKid" (transporte escolar con GPS).

CONTEXTO DE LA APP:
- Nombre: FurgoKid
- Propósito: Plataforma de transporte escolar seguro con rastreo GPS en tiempo real
- Usuarios: Padres y conductores de furgones escolares
- Colores de marca:
  * Primario: #4A90E2 (azul)
  * Secundario: #2E7BC4 (azul oscuro)
  * Acento: #FF6B6B (rojo suave)
  * Fondo: #F8F9FA (gris claro)
- Estilo: Moderno, profesional, confiable

DISEÑOS QUE NECESITO:

1. FEATURE GRAPHIC (1024x500px)
   - Para Google Play Store
   - Banner horizontal que se ve en la parte superior del listing
   - Debe incluir: Logo + Nombre "FurgoKid" + Tagline "Transporte escolar seguro"

2. SCREENSHOTS (1080x1920px) - 4 diseños
   - Para mostrar features de la app en Play Store
   - Mockups de pantallas principales
   - Con captions descriptivos

3. OPEN GRAPH IMAGE (1200x630px)
   - Para redes sociales (Facebook, Twitter, LinkedIn)
   - Se muestra cuando compartes links de la app

NECESITO QUE ME GUÍES:

PASO 1: Dame las instrucciones exactas para crear el FEATURE GRAPHIC (1024x500px):
- Qué elementos usar en Canva Pro
- Cómo configurar el tamaño exacto
- Qué texto poner y con qué fuentes
- Qué colores usar (#4A90E2, #2E7BC4)
- Qué iconos/elementos buscar en Canva Pro (bus, GPS, mapa, niños)
- Composición visual recomendada
- Instrucciones paso a paso para alguien que no es diseñador

PASO 2: Dame las instrucciones para crear 4 SCREENSHOTS (1080x1920px):
- Cómo buscar mockups de teléfono en Canva Pro
- Qué texto/captions usar para cada screenshot:
  * Screenshot 1: "Rastreo GPS en Tiempo Real"
  * Screenshot 2: "Notificaciones Instantáneas"
  * Screenshot 3: "Rutas Optimizadas"
  * Screenshot 4: "Comunicación Segura"
- Cómo hacer que se vean profesionales
- Tips de diseño para screenshots de app store

PASO 3: Dame las instrucciones para el OPEN GRAPH IMAGE (1200x630px):
- Diseño optimizado para compartir en redes
- Cómo destacar el logo y nombre
- Composición para que se vea bien en preview

IMPORTANTE:
- Dame instrucciones paso a paso MUY DETALLADAS
- Asume que tengo Canva Pro (acceso a todos los elementos premium)
- Quiero que los diseños se vean profesionales pero sin ser diseñador
- Usa términos simples (no jerga de diseño)
- Dame las medidas exactas, fuentes específicas, y ubicación de elementos
- Incluye shortcuts de Canva si ayudan

FORMATO DE RESPUESTA:
Por favor responde con:
1. Instrucciones paso a paso numeradas
2. Capturas de pantalla o descripciones de dónde hacer click
3. Valores exactos (tamaño, color hex, fuente)
4. Tips pro para que se vea bien

¿Puedes empezar con el FEATURE GRAPHIC primero?
